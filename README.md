Overview

This project is a content-based book recommendation system built with Python.
Given a book title, the system analyzes book descriptions using Natural Language Processing (NLP) techniques and recommends similar books based on textual similarity.

The recommendations are generated by comparing book descriptions using TF-IDF vectorization and cosine similarity, providing a transparent and explainable similarity score for each recommendation.

üß† How It Works

1. Data Loading (pandas)

The project uses pandas to load and manage structured book data from a CSV file:

df = pd.read_csv('books.csv')

Each row represents a book with the following fields:
	‚Ä¢	TITLE
	‚Ä¢	AUTHOR
	‚Ä¢	GENRE
	‚Ä¢	YEAR
	‚Ä¢	DESCRIPTION

Pandas allows fast indexing, filtering, and clean interaction with the dataset.

‚∏ª

2. Text Vectorization (TF-IDF)

Book descriptions are converted into numerical vectors using TF-IDF (Term Frequency‚ÄìInverse Document Frequency):

vectorizer = TfidfVectorizer(stop_words='english')
tfidf_matrix = vectorizer.fit_transform(df['DESCRIPTION'].fillna(''))

Why TF-IDF?
	‚Ä¢	Highlights important words in each description
	‚Ä¢	Downweights common words shared across many books
	‚Ä¢	Produces a meaningful vector representation of text content

Each book becomes a vector in a high-dimensional space where each dimension represents a word.

‚∏ª

3. Similarity Calculation (Cosine Similarity)

To measure how similar two books are, the system uses cosine similarity:

cosine_sim = cosine_similarity(tfidf_matrix, tfidf_matrix)

Cosine similarity measures the angle between two vectors, not their magnitude:
	‚Ä¢	1.0 ‚Üí very similar
	‚Ä¢	0.0 ‚Üí unrelated

This makes it ideal for comparing text documents of different lengths.

‚∏ª

4. Recommendation Logic

When a user inputs a book title:
	1.	The title is converted to lowercase to allow case-insensitive matching
	2.	The system finds the index of the selected book
	3.	Similarity scores between that book and all others are retrieved
	4.	Results are sorted by similarity
	5.	The top 5 most similar books are returned (excluding the original book)

Each recommendation includes a Similarity Score, calculated as:

score_percentage = round(score * 100, 2)

This transforms a mathematical similarity value into an interpretable percentage.
